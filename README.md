# pandemic
CLI tool for the smart scientist

## Install

```
npm install -g  git+ssh://git@github.sf.mpg.de:lrigoux/pandemic.git
```

> Note that you need to have a proper ssh authorization to login to github.sf.mpg.de for this to work

## Usage

```
pandemic help
```

## Requirements
- node with npm
- git
- Pandoc
- LaTeX
- Python
